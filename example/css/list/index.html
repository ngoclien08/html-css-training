Marker styles

  • animation-*: Used to apply animations to elements. The * is all animation smaller property like animation-name
  • transition-*: Used to create smooth transitions between different property values.
  • color: Specifies the text color.
  • direction: Specifies the text direction.
  • font-*: Used to specify various font properties.
  • content: Used to insert generated content before or after an element.
  • unicode-bidi: Used to control the direction of text and the position of the bullet in case of bi-directional texts (mix of rtl and ltr texts).
  • white-space: Specifies how white space inside an element is handled.
