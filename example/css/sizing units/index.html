Number

The line height is a ratio of 24px, rather than a specific size. The line height is currently 1.5, which calculates as 36px.

Percentage

This p element is 50% of the parent div's width, which is 150px. Percentage is always calculated from parent's width. Translate is calculated from p tag's width



Dimension and length

Absolute length

Absolute length is good when designing for print: cm (96px/2.54), mm, Q (1/4 of mm), in, pc (picas = 1/6 of 1in), pt (point = 1/72 of 1in), px (1/96 of 1in)

Relative length

Font-size relative length

  • 1.5em = 150% of font-size

  • 1ex = height of letter "x"

  • 1cap = height of capital letter

  • 1ch = width of "0" of a font

  • 1ic = width of longest character of a font

  • 1rem = font size of the root element (default 16px)

  • 1lh = line height of the element

  • 1rlh = line height of the root element

Viewport relative unit

  • 1vw = 1% of viewport's width

  • 1vh = 1% of viewport's height

  • 1vi = 1% of viewport's size in the root element's inline axis

  • 1vb = 1% of viewport's size in the root element's block axis

  • 1vmin = 1% of viewport's smaller dimension (if width < height then vmin = 1% width)

  • 1vmax = 1% of viewport's larger dimension

Miscellaneous units

Angle units: 60deg, .5turn

Resolution units: 192dpi (dots per inch)
